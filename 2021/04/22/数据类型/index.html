数据类型

  • list

    • append 在列表末尾追加新的元素

    • cont 统计某个元素在列表中出现的次数

    • extend 在列表的末尾一次性追加另一个序列中的多个值

    • index 从列表中找出某个值第一个匹配项的索引位置

    • insert 用于将对象插入到列表中

    • pop 移除列表中的一个元素(默认最后一个),返回该元素的值

    • remove 用于移除列表中某个值的第一个匹配项

    • reverse 方法将列表中的元素反向存放

    • sort 对列表进行排序,改变原来的列表

  • tuple

    • 比较两个值

      • com(x,y)

    • 把序列转换成列表

      • list(seq)

    • 返回序列或者参数集合中的最小值

      • min(args)

    • 返回已经排列的包含seq 所有元素的列表

      • sorted(seq)

  • dict

    • clear

      • clear方法清除字典中所有的项，这是一个原地操作，所以无返回值（或则说返回None）

        • d = {}
d['Tom']=8777             # 在字典中添加数据           
d['Jack']=9999    
print(d)                 #the result : {'Jack': 9999, 'Tom': 8777}
d.clear()
print(d)                #the result : {}

    • fromkeys

      • fromkeys方法使用给定的键建立新的字典，每个键都对应一个默认的值None。

        • print({}.fromkeys(['name','age']))         #the result　：　{'age': None, 'name': None}

    • 循环字典

      • d = {'Tom':8777,'Jack':8888,'Fly':6666}
# 方法1：
for k,v in d.items():
    print(k,v)
# 方法2
for k in d.values():
    print(k)
# 方法:3
for k in d.keys():
    print(k)

    • update

      • update方法可以利用一个字典项更新另一个字典，提供的字典中的项会被添加到旧的字典中，如有相同的键则会被覆盖

        • d = {'Tom':8777,'Jack':8888,'Fly':6666}
a = {'Tom':110,'Test':119}
d.update(a)
print(d)                        #the result ：{'Fly': 6666, 'Test': 119, 'Jack': 8888, 'Tom': 110}

  • set

    • 去重

      • list_1 = [1,2,3,4,5,1,2]
#1、去重(去除list_1中重复元素1,2)
list_1 = set(list_1)                                #去重： {1, 2, 3, 4, 5}
print(list_1)
list_2 = set([4,5,6,7,8])

    • 并集

      • #3、并集（在list_1和list_2中的元素全部打印出来，重复元素仅打印一次）
print(list_1.union(list_2))                          #并集： {1, 2, 3, 4, 5, 6, 7, 8}

    • 子集

      • #5、子集
print(list_1.issubset(list_2))                          #子集：    False    List_1中的元素是否全部在list2中

    • 在集合中添加一个元素

      • #7、在集合中添加一个元素999
list_1.add(999)
print(list_1)                                       #Add()方法：          {1, 2, 3, 4, 5, 999}

  • str

    • find

      • find方法可以在一个较长的字符串中查找子串，他返回子串所在位置的最左端索引，如果没有找到则返回-1

      • a = 'abcdefghijk'
print(a.find('abc'))                         #the result ： 0
print(a.find('abc',10,100))                    #the result ： 11  指定查找的起始和结束查找位置

    • split

      • 这是一个非常重要的字符串，它是join的逆方法，用来将字符串分割成序列

      • print('1+2+3+4'.split('+'))                            #the result ： ['1', '2', '3', '4']

    • replace

      • replace方法返回某字符串所有匹配项均被替换之后得到字符串

      • print("This is a test".replace('is','is_test'))     #the result ： This_test is_test a test

  • frozenset

    • 有序集合,不能添加和删除任何元素

  • python2和python3的区别

    • python2

      • 类需要继承(object)顶级父类

        • class test(object):

      • input输入的是str类型

      • Python3中/表示整除

  • 字符串操作match和search

    • match方法

      • re.match(): 尝试从字符串的起始位置匹配一个模式，如果不是起始位置匹配成功的话，match()就返回none

    • 正则之search

      • re.search 扫描整个字符串并返回第一个成功的匹配

    • re.match(): 语法

      • re.match(pattern, string, flags=0)

    • 匹配类型：

      • 没有次数限制的匹配为贪婪匹配
      • 有次数限制的匹配为非贪婪匹配
      • {3，} 表示最少为3次，是贪婪匹配
      • {，100}表示最多

    • 正则之findall

      • 在字符串中找到正则表达式所匹配的所有子串，并返回一个列表，如果没有找到匹配的，则返回空列表
pattern 匹配的正则表达式
string 要匹配的字符串。
flags 标志位，用于控制正则表达式的匹配方式，如：是否区分大小写，多行匹配等等。 
                 匹配成功**re.match**方法返回一个匹配的**对象**，否则返回None。

正则匹配单字符

字符 功能
. 匹配任意1个字符（除了\n）
[ ] 匹配[ ]中列举的任意一个字符
\d 匹配单个数字，即0-9
\D 匹配非数字，即不是数字
\s 匹配空白，即 空格
\S 匹配非空白
\w 匹配单词字符，即a-z、A-Z、0-9、_
\W 匹配非单词字符

匹配多个字符

字符 功能
* 匹配前一个字符出现0次或者无限次，即可有可无
+ 匹配前一个字符出现1次或者无限次，即至少有1次
? 匹配前一个字符出现1次或者0次，即要么有1次，要么没有
{m} 匹配前一个字符出现m次
{m,n} 匹配前一个字符出现从m到n次

  • 深拷贝deepcopy: 深拷贝会完全复制原变量相关的所有数据，在内存中生成一套完全一样的内容，我们对这两个变量中任意一个修改都不会影响其他变量
